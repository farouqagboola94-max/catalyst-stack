# /voice-to-content

---
description: Convert a voice note transcript or raw spoken text into 3 formatted content pieces — Substack article, YouTube script, and 60-second Shorts script.
---

## Instructions

You will receive either:
- A raw voice note transcript (messy, spoken language, filler words included)
- A topic/theme to develop from scratch

### Step 1 — Clean the Transcript
If a transcript is provided:
- Strip filler words (um, uh, like, you know)
- Preserve the speaker's natural rhythm and ideas
- Extract the core argument or insight

### Step 2 — Identify the Core Thesis
One sentence. The single most powerful idea in the content. This becomes the hook.

### Step 3 — Generate Format 1: Substack Article
- Length: 800-1200 words
- Structure: Hook → Context → The Insight → The Implication → Close
- Voice: Direct, contraction-heavy, no corporate language
- Start with a punchy first line that makes the reader feel something
- No headers unless the piece is genuinely sectioned
- Lagos perspective where it fits naturally

### Step 4 — Generate Format 2: YouTube Script
- Length: 6-10 minutes read time (900-1500 words)
- Structure: Hook (0-30s) → Problem setup → Core insight → Examples → Call to action
- Include [PAUSE] markers and [B-ROLL: description] cues
- Conversational but authoritative
- End with a specific CTA (subscribe, comment a word, check Substack)

### Step 5 — Generate Format 3: 60-Second Shorts Script
- Length: 120-150 words max
- Structure: Hook (first 3 seconds) → Rapid fire points → Punchline close
- Written to be spoken fast, with urgency
- Include [CUT] markers between beats
- First line must stop the scroll

### Step 6 — Output Package
Deliver all three formats clearly labeled. Then add a "Repurpose Notes" section with 3 caption angles for Instagram and Twitter pulled from the piece.

## Output Format

```
CORE THESIS: [one sentence]

---
FORMAT 1: SUBSTACK
[full article]

---
FORMAT 2: YOUTUBE SCRIPT
[full script with cues]

---
FORMAT 3: 60-SECOND SHORTS
[short script with cut markers]

---
REPURPOSE NOTES
IG Caption Option 1: [caption]
IG Caption Option 2: [caption]
Twitter/X Thread Hook: [opening tweet]
```
