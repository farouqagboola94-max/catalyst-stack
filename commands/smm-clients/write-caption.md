# /write-caption

---
description: Write a platform-optimized social media caption for any client post. Fast, on-brand, ready to post. Specify client, platform, and post context.
---

## Instructions

### Step 1 — Input Requirements
If not provided in the prompt, ask for:
1. **Client/brand** — Who is this caption for?
2. **Platform** — Instagram, Facebook, Twitter/X, LinkedIn, TikTok
3. **Post context** — What is being posted? (product image, reel, announcement, quote graphic, etc.)
4. **Key message** — What should the post communicate?
5. **CTA** — What action should the viewer take?

### Step 2 — Platform Rules

**Instagram:**
- Max 2,200 characters but keep body under 150 before "more" cut
- Line breaks matter — short punchy lines
- Hook in the FIRST LINE
- Hashtags at the bottom, 8-15 tags
- One clear CTA

**Facebook:**
- More conversational, can be longer
- Questions work well to drive comments
- Hashtags: 2-3 only
- Link previews auto-populate — mention the link context

**Twitter/X:**
- 280 characters per tweet
- If longer, structure as thread hook
- No hashtag spam — 1-2 max
- Hook must be standalone if retweeted

**LinkedIn:**
- Professional but not corporate
- Short paragraphs, lots of line breaks
- Personal angle or business insight
- 2-3 hashtags max
- No fluff — every sentence earns its place

**TikTok:**
- Caption is secondary — focus on hook line
- 150 characters ideal
- 3-5 relevant hashtags
- Include call to save/comment/stitch if relevant

### Step 3 — Write 3 Caption Variations
Always deliver 3 options:
- **Option A** — Direct/Informational
- **Option B** — Story/Emotional angle
- **Option C** — Bold/Provocative (pushes the edge of the brand voice)

Let the client choose.

### Step 4 — Hashtag Research
Provide a hashtag block tailored to the post. Mix:
- High volume (1M+): 2-3 tags
- Medium volume (100K-1M): 5-7 tags
- Niche/brand (under 100K): 3-5 tags

## Output Format

```
CLIENT: [NAME] | PLATFORM: [PLATFORM] | POST TYPE: [TYPE]

OPTION A — Direct:
[caption]
[hashtags]

OPTION B — Story:
[caption]
[hashtags]

OPTION C — Bold:
[caption]
[hashtags]

RECOMMENDED: Option [X] — [reason in one sentence]
```
