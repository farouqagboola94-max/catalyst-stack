# /smm-report

---
description: Generate a monthly social media performance report for an SMM client. Takes raw metrics input and produces a clean, branded report ready to send to the client. Includes insights and next month's recommendations.
---

## Instructions

### Step 1 — Data Input
Ask for the following metrics (or accept them if pasted):

**Account Overview:**
- Follower count (start of month)
- Follower count (end of month)
- Net follower growth

**Post Performance:**
- Total posts published
- Total reach
- Total impressions
- Average engagement rate
- Top 3 performing posts (content type, reach, engagement)
- Bottom 3 posts (content type, reach, engagement)

**Stories/Reels (if applicable):**
- Number of stories posted
- Average story views
- Number of Reels posted
- Average Reel views, plays, shares

**Website/Link Traffic (if tracked):**
- Link in bio clicks
- Website sessions from social

**DMs/Inquiries:**
- Number of DMs received
- Number of leads or inquiries generated

### Step 2 — Calculate Key Metrics

- **Follower growth rate:** (New followers / Start count) x 100
- **Average engagement rate:** (Likes + Comments + Shares + Saves) / Reach x 100
- **Best performing content type:** Based on top 3 posts data
- **Worst performing content type:** Based on bottom 3 posts data

### Step 3 — Write the Report

Structure:

```
[CLIENT NAME] — SOCIAL MEDIA REPORT
[MONTH] [YEAR]
Prepared by: Catalyst OS | SMM Division

---
EXECUTIVE SUMMARY
[3-4 sentence overview of the month — growth, wins, areas to address]

---
KEY METRICS AT A GLANCE
| Metric | This Month | Last Month | Change |
[table of core numbers]

---
AUDIENCE GROWTH
[Follower stats with growth commentary]

---
CONTENT PERFORMANCE
[Top performers analysis — what worked and why]
[Bottom performers analysis — what didn't and why]

---
WHAT'S WORKING
[3 specific observations backed by data]

---
WHAT TO ADJUST
[3 specific recommendations with rationale]

---
NEXT MONTH'S FOCUS
[3 content/strategy priorities for Month+1]

---
NOTES
[Any client-specific context, events coming up, etc.]
```

### Step 4 — Tone
- Professional but clear
- No jargon the client won't understand
- Data-first — every claim has a number behind it
- Honest about what didn't work — don't sugarcoat, explain it

## Output Format

Clean report text ready to paste into a Word document or PDF. Use the structure above exactly.
