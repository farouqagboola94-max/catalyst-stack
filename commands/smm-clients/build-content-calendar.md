# /build-content-calendar

---
description: Build a 30-day social media content calendar for an SMM client. Platform-specific, on-brand, and ready to hand off for scheduling. Works for any client — prompts for brand context before building.
---

## Instructions

### Step 1 — Client Brief (If Not Provided, Ask)

1. **Client name/brand**
2. **Industry/niche** — e.g., Aso Oke fashion, food, real estate, lifestyle
3. **Primary platform(s)** — Instagram, Facebook, Twitter/X, LinkedIn, TikTok
4. **Posting frequency** — e.g., 5x/week, daily, 3x/week
5. **Brand voice** — Formal, casual, luxury, energetic, cultural, educational
6. **Target audience** — Demographics, interests, location
7. **Content pillars** — 3-5 themes the brand posts about (e.g., product, lifestyle, education, behind-the-scenes, social proof)
8. **Upcoming campaigns or events** — Any launches, sales, or dates to work around
9. **CTA preference** — Shop now, DM us, link in bio, comment below, etc.
10. **Month** — Which month is this calendar for?

### Step 2 — Build the Calendar

Structure: Table format, 30 days, columns:

| Day | Date | Platform | Content Pillar | Post Type | Caption Direction | Visual Direction | CTA |

**Post Types:**
- Static image
- Carousel (2-10 slides)
- Reel/Short video
- Story
- Text post (Twitter/LinkedIn)
- UGC repost

**Distribute content types:** At least 30% video/Reels. No two identical pillar days in a row.

### Step 3 — Write 5 Full Captions
From the 30 days, pick 5 high-priority posts and write the full caption. Include hashtag block (max 15, research-relevant).

Caption structure:
- Line 1: Hook (stop the scroll)
- Lines 2-4: Body (value, story, or product highlight)
- Line 5: CTA
- Hashtags (grouped separately, not buried in caption)

### Step 4 — Visual Direction Notes
For each of the 5 full captions, add a visual brief:
- Color mood
- Shot type (flat lay, lifestyle, close-up, model, graphic)
- Props or setting
- Text overlay needed? (Yes/No + suggested copy)

## Output Format

```
CLIENT: [NAME]
MONTH: [MONTH/YEAR]
PLATFORMS: [LIST]
POSTING FREQUENCY: [X/week]

30-DAY CALENDAR:
[Table]

---
5 FULL CAPTIONS WITH VISUAL BRIEFS:

POST [DATE] — [PILLAR]:
Caption: [full caption]
Hashtags: [hashtag block]
Visual Brief: [direction]

[Repeat x5]
```
