## Setup (Do This Once)

```bash
npm install -g @anthropic-ai/claude-code
cd catalyst-os
claude
```

That's it. Claude Code reads the SKILL.md files and CLAUDE.md automatically.

## Quick Command Reference

### Content Pipeline
```bash
claude /voice-to-content       # Voice note → 3 content formats
claude /substack-draft         # Draft next Great Sabotage part
claude /repurpose-content      # One piece → LinkedIn + Twitter + IG caption
```

### Chambers Legal
```bash
claude /draft-affidavit        # Affidavit of Service
claude /draft-particulars      # Particulars of Claim
claude /format-legal-doc       # Format any raw legal text
claude /chambers-pipeline      # Full: instruction → formatted + filed doc
```

### SMM Clients
```bash
claude /build-content-calendar # 30-day calendar for a client
claude /write-caption          # On-brand caption for any post
claude /smm-report             # Monthly performance report
```

### Sneakers Fest
```bash
claude /sneakers-fest-promo    # Drop content for the event
claude /vendor-comms           # Vendor outreach and follow-up emails
claude /event-brief            # One-page event brief document
```

### Agent Teams (Run Everything)
```bash
claude /run-content-week       # Full week of content, all platforms
claude /run-chambers-batch     # Process all pending legal docs
```

## Your Content Voice Rules (Auto-applied)

Every output from this system follows these rules:
- Direct, contraction-heavy (don't, can't, won't)
- No hyphens between words. Spaces only.
- No em-dashes
- No "Imagine this," "In today's landscape," or corporate buzzwords
- Every piece starts with a punchy high-stakes hook
- Lagos-grounded perspective where relevant
