1. Culture — Why this event matters to Lagos

## Instructions

### Step 1 — Identify Content Phase
Determine which phase of the event cycle this content is for:

- **Pre-event (Announcement)** — Building awareness, early excitement
- **Pre-event (Countdown)** — Creating urgency, vendor spotlights
- **Pre-event (Week of)** — Final push, logistics info, hype
- **Day-of** — Live content, real-time updates
- **Post-event (Recap)** — Highlights, numbers, next year tease

### Step 2 — Specify Format Needed
If not specified, generate for Instagram first, then adapt.

**Format options:**
- Caption only
- Caption + carousel slide scripts
- Caption + Reel hook/script
- Twitter/X thread
- WhatsApp broadcast (for vendor/community group)
- Email newsletter

### Step 3 — Write the Content

**Voice for Sneakers Fest:**
- Lagos energy — direct, cultural, not over-produced
- Reference Lagos landmarks and culture where natural
- Hype without desperation — the event speaks for itself
- FOMO is the engine, community is the fuel

**Every piece must include:**
- Hook that creates either excitement or FOMO
- Event specifics where relevant (date, location, vendors)
- Clear CTA (register, DM, follow, share)
- Hashtag block: #SneakersFest2026 #LagosSneakers #[others]

### Step 4 — Deliver 2 Versions
Always give 2 options:
- **Hype version** — High energy, punchy
- **Culture version** — More depth, community-focused
