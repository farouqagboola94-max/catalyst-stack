## What this map kills

Three things this map kills, before they happen:

1. **Tool sprawl**: Every tool has a slot, a tier, and a revenue stream. No tool joins the stack without earning placement.
2. **Duplicate spending**: Portkey vs OmniRoute, n8n vs Huginn, Cursor vs Windsurf. Pick one per slot.
3. **Drift from revenue**: Every DEPLOY_NOW tool maps to a revenue stream. If a tool produces no money this quarter, it does not deploy this quarter.
