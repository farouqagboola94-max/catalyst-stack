# PRIORITY TIERS // CATALYST OS

The arsenal's 60 tools are graded against one filter: does this tool move a Lagos based five stream consulting and content operation forward this quarter, next quarter, or never?

## The four tiers

### DEPLOY_NOW (29 tools)

Tools that produce immediate operational value. Already integrated, or worth integrating this quarter.

The cluster:
- Claude Code, Superpowers, pydantic-ai, FastAPI (foundation, already running)
- Tavily, Context7, Task Master AI, MCP Playwright, fastmcp, markdownify-mcp (MCP layer, deploy this month)
- PDF Processing, Frontend Design, Skill Creator, Marketing Skills, Obsidian Skills, Context Optimization, Deep Research (active skill layer)
- OpenClaw, CrewAI, Dify (agent layer for Phase 1 hires)
- n8n (automation backbone)
- GPT Researcher, Firecrawl, Instructor, Chroma, ExtractThinker (data and research layer)
- Codebase Memory MCP (CatalystOS continuity)
- Awesome Claude Skills, Anthropic Skills Repo, Anthropic Prompt Eng Tutorial, SkillsMP, Anthropic Official Docs (reference set)

### Q3_EXPANSION (16 tools)

Tools that activate when the Phase 1 team is hired, Sneakers Fest production scales, or Chambers AI Phase 2 begins. Track these. Do not deploy yet.

The cluster:
- Spec Kit, LangGraph, OWL, CopilotKit
- Ollama, Open WebUI
- Langflow, Temporal
- Claude SEO

Reference layer. Skim quarterly. Useful for AI Tech VA onboarding curriculum or one off comparison testing.

### SKIP (3 tools)

Out of scope. GPU intensive fine tuning and production model serving are not relevant to a Lagos consulting operation.

The cluster:
- Unsloth
- vLLM
- (Plus any future GPU heavy deployment tools that show up in MAGI//ARCHIVE)

## Reassessment cadence

Run the tier audit quarterly:
- Q3 2026: Promote tools from Q3_EXPANSION to DEPLOY_NOW as Phase 1 team activates.
- Q4 2026: Audit DEPLOY_NOW for tools that did not earn their slot. Demote ruthlessly.

## The filter, written plain

A tool earns DEPLOY_NOW if and only if Catalyst, working alone or with the current contractor pool, will use it this quarter to ship something a client pays for or that advances a public content artifact.

Everything else waits.
