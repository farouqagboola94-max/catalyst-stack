How to Deploy These Skills
These are Claude Code skills — they work inside the Claude Code CLI. Here's the 4-step activation protocol.
STEP 01
Install Claude Code
Install via npm: npm install -g @anthropic-ai/claude-code. Requires Node.js. Run claude to initialize your project.
STEP 02
Get the Skill Repos
Clone from GitHub. Priority first: Brand Guideline and Marketing Skills from anthropics/skills and coreyhaines31/marketingskills.
STEP 03
Add to Your Project
Place the SKILL.md files into your Claude project context. Claude Code reads them automatically and applies their protocols to your workflow.
STEP 04
Build Custom Skills
Use SS-10 (Skill Creator) to build your own. Describe your voice note pipeline workflow and generate a permanent SKILL.md for it.
