# Catalyst OS: MCP Intelligence Carousel

> Part of the **Catalyst OS** ecosystem. Lagos Noir aesthetic. Built for operators, not spectators.

## What This Is

An interactive carousel component showcasing the **Top 5 MCPs (Model Context Protocol servers) that 7 figure businesses can't operate without**. Built as a React component in the Catalyst OS design language (dark backgrounds, orange accents, geometric precision).

### The 5 MCPs

| # | MCP | What It Does |
|---|-----|-------------|
| 01 | **Shopify MCP** | Manage your store, inventory, orders, and products via plain English |
| 02 | **Higgsfield MCP** | Generate AI images, videos, and creative content at scale |
| 03 | **AdAdvisor MCP** | Run, analyze, and optimize Meta ads without opening Ads Manager |
| 04 | **Google Drive MCP** | Search, summarize, and organize files across your entire Drive |
| 05 | **Google Analytics MCP** | Track performance, discover insights, and understand user behavior |

## Stack

- React (functional components, hooks)
- Tailwind free (all inline styles for zero dependency portability)
- Touch/swipe support for mobile
- Keyboard navigation (arrow keys)
- Lagos Noir design system: `#0a0a0a` base, `#f97316` accent

## Integration

Drop into any React project:

```jsx
import MCPCarousel from './src/MCP_Carousel';

function App() {
  return <MCPCarousel />;
}
```

## Catalyst OS Module

This component is registered as the **MCP Intelligence Module v1.0** within Catalyst OS. It integrates with:

- The Catalyst Skill Vault (AI Consulting OS pack)
- The Switchboard module routing system
- The main CatalystOS React dashboard
